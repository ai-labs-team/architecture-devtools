# Axiom Architecture DevTools

## Installation

 * `yarn`
 * `yarn build`
 * Open [chrome://extensions](chrome://extensions)
 * Make sure `[✔] Developer Mode` is ✔'d
 * Click `Load unpacked extension...`
 * Select this directory
 * If DevTools is open, close and reopen it
